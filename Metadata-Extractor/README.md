# 🖼️ Metadata Extractor

This script scans a directory for **image files** and displays their metadata in a table.  
It includes filename, extension, format, dimensions, and color mode for each image.

## ✨ Features

- Recursively scans directories for images  
- Displays metadata: filename, extension, format, width, height, mode  
- Outputs results in a clean table format using PrettyTable  

## ⚙️ Requirements

- Python 3.x  
- Libraries: `Pillow`, `prettytable`  

Install dependencies:

```bash
pip install Pillow prettytable
