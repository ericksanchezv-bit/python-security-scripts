These projects explore cybersecurity concepts such as digital forensics, network analysis, data encoding, and ethical hacking techniques using Python.

## 🔍 Overview

Each script demonstrates practical cybersecurity and scripting techniques including:
- Parsing and analyzing log files
- Extracting and hashing forensic data
- Capturing and inspecting network packets
- Implementing socket-based communication
- Exploring covert data transmission methods

These scripts are strictly for **educational and ethical use** as part of university coursework.

## 📜 Projects Included

| Project | Description |
|----------|--------------|
| [File Processor](file_processor/file_processor.py) | Extracts and displays file metadata and headers |
| [Firewall Log Parser](firewall-log-parser)| Parses firewall logs for patterns and anomalies |
| [Forensic Hashing Tool](Forensic-Hashing-Tool)| Performs SHA-256 hashing for file integrity verification |
